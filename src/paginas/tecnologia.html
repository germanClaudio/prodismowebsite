Tecno Software

AUTOFORM

Las soluciones del software AutoForm forman una plataforma integral para la Ingeniería, la evaluación y la mejora de los procesos de conformado de chapa metálica y ensamblaje de BIW. Diseñadas para las aplicaciones de la vida real, nuestras Soluciones incrementan el desempeño de la ingeniería y garantizan la prroducción óptima de cada pieza.

E-PLANNER SOFTWARE

Programa específico de Ingeniería Eléctrica, o CAE. Asiste el diseño de esquemas eléctricos y neumáticos, optimizando y automatizando al máximo el proceso de desarrollo de un proyecto eléctrico. Proporciona bases de datos con información sobre el proyecto, BOM, relación in/out salidas PLC, borneras y conexiones, secuencias entre otras.

CATIA SOFTWARE

Software para el Diseño Mecánico CAD de Dispositivos y Máquinas Especiales. El programa proporciona apoyo desde la concepción del diseño hasta la producción y análisis del producto. Posibilita la creación de piezas en 3D, 2D, Cálculos de diferentes materiales, manejo y creación de superficies, creación de listas de materiales automáticas, todo 100% vinculado con un único diseño. Aquí se defininen tolerancias, cinemáticas, materiales necesarios para la construcción de cada elemento y ensambles.

PROCCESS SIMULATE

Software que permite realizar la simulación mecánica completa de nuestras líneas de ensamble. Permite diseñar espacios, dimensionar equipos y validar la accesibilidad y espacios de colisiones entre Robots / Máquinas / Dispositivos, como así también definir y extraer trayectorias y secuencias de funcionamiento de los mismos. Herramienta fundamental para diseñar y comprobar factibilidad y viabilidad de procesos de líneas de ensamble.
